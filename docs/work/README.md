# Work Notes Archive

`docs/work/**` contains planning artifacts, evaluator notes, design drafts, and
historical research created while Forge was being built. These files are useful
for maintainers who need the decision trail, but they are not the canonical user
documentation for v0.0.11.

For current public guidance, start with:

- [README](../../README.md)
- [Quickstart](../../QUICKSTART.md)
- [Documentation index](../INDEX.md)
- [Workflow templates](../guides/WORKFLOW_TEMPLATES.md)
- [Skills and command projections](../reference/SKILLS.md)
- [Support guide](../guides/SUPPORT.md)

Historical files may mention old internal labels, earlier path conventions, or
draft commands. Treat those as implementation history unless a current guide or
reference page confirms the behavior.
